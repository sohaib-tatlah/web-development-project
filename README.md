# Assignment 1 – Landing Page Clone (Plain CSS Only)

## 🎯 Project Overview
This is a professional E-commerce landing page inspired by Sveston Watches, built using **HTML5** and **Plain CSS only**. No frameworks (Bootstrap, Tailwind) or JavaScript were used.

## 📁 Project Structure
```
assignment-1/
├── index.html          # Main HTML file with semantic structure
├── style.css           # Complete CSS styling with responsive design
├── assets/             # Folder for images (currently using placeholder gradients)
└── README.md           # Project documentation
```

## ✨ Features Implemented

### 1. Semantic HTML5 Structure
- ✅ `<header>` with navigation
- ✅ `<nav>` menu
- ✅ `<section>` for different content areas
- ✅ `<main>` for primary content
- ✅ `<footer>` with multiple columns

### 2. Header Section
- ✅ Logo on the left (SVESTON)
- ✅ Navigation menu (Home, Men, Women, Collections, Sale, Contact)
- ✅ Icons for search, user, cart
- ✅ Clean horizontal layout using Flexbox
- ✅ Sticky header with smooth shadow

### 3. Hero Section
- ✅ Full-width banner with background image
- ✅ Large heading text ("Timeless Elegance")
- ✅ Short tagline
- ✅ "Shop Now" button with hover effect
- ✅ Centered content

### 4. Featured Categories Section
- ✅ 4 category cards (Men's, Women's, Sport, Luxury)
- ✅ Each card contains image and title
- ✅ Cards aligned in a row using CSS Grid
- ✅ Smooth hover effects with transform

### 5. Product Section (Best Sellers)
- ✅ Grid layout of 4 products per row
- ✅ 8 product cards total
- ✅ Each card includes:
  - Product Image (gradient placeholder)
  - Product Name
  - Price
  - Add to Cart button
- ✅ Card shadow effects
- ✅ Button hover styling

### 6. Promotional Banner Section
- ✅ Background image with overlay
- ✅ Promotional text ("Limited Time Offer")
- ✅ Centered button with hover effect

### 7. Footer
- ✅ 4 columns:
  - About Sveston
  - Customer Service
  - Quick Links
  - Follow Us (Social Media)
- ✅ Copyright text at bottom
- ✅ Responsive grid layout

## 🎨 Styling Features

### CSS Techniques Used
- ✅ **Flexbox** for header and navigation layout
- ✅ **CSS Grid** for categories, products, and footer
- ✅ **Hover effects** on all interactive elements
- ✅ **Transitions** for smooth animations
- ✅ **Box shadows** for depth and elevation
- ✅ **Transform** properties for hover effects
- ✅ **Google Fonts** (Playfair Display + Poppins)

### Color Palette
- Primary: `#1a1a1a` (Black)
- Accent: `#c9a961` (Gold)
- Background: `#fff` (White)
- Secondary BG: `#f8f8f8` (Light Gray)
- Footer: `#1a1a1a` (Dark)

## 📱 Responsive Design

### Mobile View (max-width: 768px)
- ✅ Navbar converts to vertical layout

- ✅ Product grid: **2 per row**
- ✅ Category cards: **1 per row** (stacked vertically)
- ✅ Adjusted font sizes

- ✅ Footer stacks into single column

### Tablet View (max-width: 1024px)

- ✅ Category grid: **2 per row**

## 🚀 How to Use

1. **Open the Project**
   - Navigate to the `assignment-1` folder
   - Open `index.html` in any modern web browser

2. **View on Different Devices**
   - Desktop: Full 4-column layout
   - Tablet: 3-column product grid
   - Mobile: 2-column product grid, stacked categories

3. **Test Responsive Design**
   - Press `F12` in your browser
   - Click the device toolbar icon
   - Test different screen sizes
